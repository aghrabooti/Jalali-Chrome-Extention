# Jalali-Chrome-Extention

This project is a simple and clean implementation of a Persian (Jalali) calendar using **HTML**, **CSS**, and **JavaScript**.  
It is designed to provide an easy-to-use, RTL-compatible interface for Persian-speaking users.

---

## Features

- Displays the current **Jalali (Shamsi)** date and month.  
- Fully **right-to-left (RTL)** layout.  
- Simple and minimal user interface with traditional color tones.  
- Uses [`jalaali-js`](https://github.com/jalaali/jalaali-js) for accurate date conversion.  
- Upcoming features:
  - Reminder functionality for adding and managing small daily alerts.  
  - A built-in to-do list for personal note-taking.
